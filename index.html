Atlas World Cleaner

Informations

Note! The ZIP file must contain the world files directly. Do not upload an archive that includes an extra folder layer (such as "world"). The archive itself should be the world, not a container for it.

What does this site do?
It removes empty (0 KB) files that can prevent world imports on Atlas. It also resets player data so that existing players do not carry over any progression into the instance that will be uploaded to Realms, including advancements, inventories, and statistics.
